# F-lab 멘토링
> 현직자 멘토와의 멘토링 과정에서 학습한 내용을 정리한 내용을 정리한다.
> + 주차별 지정된 토픽에대해 깊게 조사하며 학습한 내용을 마크다운으로 기록한다.
> + 기록한 내용을 바탕으로 멘토와 토론한 후 수정 보완하여 상위 디렉토리의 주제별 기록으로 이동한다.

<i>내용에 대해 조사할 땐 무조건 깊게 들어가도록 노력한다! 들어간 시간과 비용을 극한까지 활용하는것을 목표로 하자.</i>
